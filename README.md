# Telegram-бот для мониторинга сервера

Учебный проект в рамках подготовки к позиции AI-DevOps стажёра.

## Возможности

| Команда  | Описание                     |
|----------|------------------------------|
| `/start` | Приветствие и список команд  |
| `/status`| Uptime сервера               |
| `/disk`  | Свободное место на диске     |
| `/cpu`   | Загрузка процессора          |
| `/help`  | Информация о проекте         |

## Стек технологий

- **Python 3.11** — язык разработки
- **python-telegram-bot** — работа с Telegram API
- **Docker** — контейнеризация
- **GitHub Actions** — CI/CD (автоматическая сборка)
- **Linux** — среда исполнения
